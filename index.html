How to use

1. Source

Paste a direct link to any .svg file. The canvas updates instantly.

2. Animate

Select a style. Standard is for UI; Advanced uses 3D & Filters for hero sections.

3. Trigger

Choose Loop for constant motion or Hover for interactive feedback.

4. Deploy

Copy the snippet. It's 100% self-contained—no hosting or JS libraries needed.

Pro Tip

Use "Checkerboard" theme to verify your SVG transparency before exporting.
